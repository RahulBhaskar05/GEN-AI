# Generative AI Challenge 2026

This repository contains solutions for the 2026 Generative AI Challenge, covering three tasks that explore technical reconstruction, structured design, and creative storytelling.

---

## Challenge 1: Image Repair + Style Transformation
- **Type:** Core Technical
- **Objective:** Reconstruct intentionally degraded images using generative models and transform them into a new visual style while preserving structure.
- **Deliverables:** Degraded images, repaired images, stylized final images, explanation of methods.

---

## Challenge 2: AI Fashion Designer
- **Type:** Technical Creative
- **Objective:** Train a generative model on a fashion dataset to create a coherent collection of 3–5 outfit designs.
- **Deliverables:** Generated outfit images, mood board or style sheet, explanation of creative process.

---

## Challenge 3: AI Poster Designer
- **Type:** Core Creative
- **Objective:** Use generative AI to design posters for a fictional concept, refining initial ideas into a polished final output.
- **Deliverables:** 2–3 initial poster concepts, 1 final poster, short rationale explaining design decisions.
